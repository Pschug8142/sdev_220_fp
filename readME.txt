Once a venv has been established, do `pip install requests`. While this module is included in the requirements.txt, it isn't automatically pulled in by django and therefore must be installed manually.

Once the django server has been launched, open the locally hosted site. From there, add /appevent/ to
the path in the browser URL to access the site.

There are 3 users in the Database: admin, Graz, and Jean. The password for all of these users is "bobbybebop"

Input a character before clicking on an event.

If adding in toon info, the page will not redirect itself to the events page, and must therefore be done manually.
